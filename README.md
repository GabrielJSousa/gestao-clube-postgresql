Base de Dados para Gestão de Clube Desportivo

Este projeto consiste na modelação e implementação de uma base de dados relacional desenhada para a gestão de um clube desportivo. O projeto foi desenvolvido no âmbito académico.

  Estrutura e Entidades
A base de dados foi projetada para suportar as operações diárias do clube, incluindo o registo e gestão das seguintes áreas:
Sócios e Funcionários: Gestão de dados pessoais e operacionais.
Aulas e atividade: Agendamento e alocação de recursos.
Mensalidades: Controlo financeiro e de pagamentos.
Stock: Gestão de inventário e equipamentos do clube.
